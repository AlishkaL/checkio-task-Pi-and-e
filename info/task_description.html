Given the lengths of the sides of a triangle. Need to find angles for this triangle. If with given sides can not be formed a triangle, then return all angles as 0 (zero). The angles should be represented as a list of integers in ascending order. Each angle is measured in degrees and rounded an nearest integer number (Standard mathematical rounding).

Input: The lengths of the sides of a triangle. Integers.

Output: Angles of a triangle in degrees. A sorted list of integers.

Example: 

checkio(4, 4, 4) == [60, 60, 60]
checkio(3, 4, 5) == [37, 53, 90]
checkio(2, 2, 5) == [0, 0, 0]
