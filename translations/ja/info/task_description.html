あなたには三角形の三辺の長さが与えられる。 この三角形の三つの角の角度をもとめること。 もしその三辺では三角形を作れない場合 (または縮退三角形になる場合)は全ての角度を0で返すこと。 角度は昇順の整数のリストで表し、 それぞれの角は度数で測り少数点以下は四捨五入すること。 (標準的な数学的四捨五入を行う)

入力: 三角形の三辺の長さ。(integer型)

出力: 三つの角の度数(integer型のリスト。※昇順)

例:

{% if interpreter.slug == "js-node" %} 
triangleAngles(4, 4, 4) == [60, 60, 60]
triangleAngles(3, 4, 5) == [37, 53, 90]
triangleAngles(2, 2, 5) == [0, 0, 0]
{% else %} 
checkio(4, 4, 4) == [60, 60, 60]
checkio(3, 4, 5) == [37, 53, 90]
checkio(2, 2, 5) == [0, 0, 0]
{% endif %}

どのように使われるか: これは古典的な幾何学の問題である。 この手法は地形学や建築で利用される。 あなたはこれにより分度器を使用せずに角度を測ることが可能である。

事前条件:
0 < a,b,c ≤ 1000
