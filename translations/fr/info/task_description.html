Les longueurs des trois côtés d'un triangle vous sont communiquées. Il vous faut déterminer les trois dangles de ce triangle. Si les longueurs communiquées ne peuvent pas être celles d'un triangle (ou formeraient un triangle dégénéré), alors retournez les trois angles à 0 (zero). Les angles seront représentés comme une liste d'entiers par ordre croissant. Chaque angle est mesuré en degrés et arroudi à l'entier le plus proche (arrondi mathématique standard).

Input: Les longueurs des côtés d'un triangle, comme entiers (int).

Output: Les angles en degrés du triangle, comme liste triée d'entiers.

Exemple:

{% if interpreter.slug == "js-node" %} 
triangleAngles(4, 4, 4) == [60, 60, 60]
triangleAngles(3, 4, 5) == [37, 53, 90]
triangleAngles(2, 2, 5) == [0, 0, 0]
{% else %} 
checkio(4, 4, 4) == [60, 60, 60]
checkio(3, 4, 5) == [37, 53, 90]
checkio(2, 2, 5) == [0, 0, 0]
{% endif %}

Application: C'est une question classique de géométrie. Ces connaissances sont précieuses en architecture et en topologie. On peut ainsi connaître la mesure d'un angle sans rapporteur.

Préconditions:
0 < a,b,c ≤ 1000
